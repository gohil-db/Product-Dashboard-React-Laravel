
### First create Backend in Laravel
✅ 1. Create Project
laravel new backend
cd backend
php artisan serve

✅ 2. Setup Database
DB_DATABASE=your_db_name
DB_USERNAME=root
DB_PASSWORD=

✅ 3. Install Sanctum (Token Auth)
composer require laravel/sanctum
php artisan vendor:publish --provider="Laravel\Sanctum\SanctumServiceProvider"
php artisan migrate

✅ 4. Configure Sanctum
In app/Models/User.php:
use Laravel\Sanctum\HasApiTokens;
class User extends Authenticatable
{
    use HasApiTokens;
}

✅ 5. Create Auth Controller
php artisan make:controller API/AuthController

✅ 6. Write Auth Logic
📌 app/Http/Controllers/API/AuthController.php

✅ 7. Add API Routes
📌 routes/api.php

✅ 8. Test API (Thunder Client / Postman)
🔹 Register
POST /api/register
{
  "name": "Dharmendra",
  "email": "test@gmail.com",
  "password": "123456"
}

🔹 Login
POST /api/login
{
  "email": "test@gmail.com",
  "password": "123456"
}
